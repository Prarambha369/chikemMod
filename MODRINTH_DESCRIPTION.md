# Chikem Mod

A comprehensive Fabric mod for Minecraft 1.21 that enhances your gameplay experience with custom features and mechanics.

## 🎮 Features

- **Custom Items**: Unique items with special properties and abilities that expand your gameplay options
- **Enhanced Mechanics**: Improved gameplay systems and interactions for a better Minecraft experience
- **Performance Optimized**: Built with efficiency in mind to ensure smooth gameplay without lag
- **Fully Configurable**: Customize features to match your playstyle and preferences
- **Multiplayer Compatible**: Works seamlessly on both servers and single-player worlds

## 📋 Requirements

- **Minecraft**: 1.21
- **Fabric Loader**: 0.16.0 or higher
- **Fabric API**: Latest version (automatically included)
- **Java**: 17 or higher

## 🔧 Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21
2. Download Fabric API from [Modrinth](https://modrinth.com/mod/fabric-api) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
3. Download Chikem Mod from this page
4. Place both JAR files in your `.minecraft/mods` folder
5. Launch Minecraft with the Fabric profile

## ⚙️ Configuration

Configuration files are automatically generated in `.minecraft/config/chikem-mod/` when you first launch the game with the mod installed. You can customize various features to suit your preferences.

## 🎯 Probability & Loot Rates

### 🐣 Special Chickem breeding chance
When **both parent chickens are fed GolChick Food** before breeding:

- **Peaceful**: **30%**
- **Easy**: **24%**
- **Normal**: **18%**
- **Hard**: **14%**
- **Hardcore**: **12%**

### 💎 Natural treasure rates
- **GolChick Food**
  - Structure treasure chests: **35%**
  - Fishing treasure: **8%**
- **Goldie Egg**
  - Structure treasure chests: **12%**
  - Fishing treasure: **3%**

Structure pools include: simple dungeon, abandoned mineshaft, desert pyramid, jungle temple, stronghold corridor, ancient city, end city treasure, and bastion treasure.
